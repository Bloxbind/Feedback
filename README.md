# Feedback Repository

Welcome! This repository is dedicated to **feature requests** and **bug reports** only.

##  Before You Submit

**Please search existing issues first!** Your feedback might already be reported. Check:
- [Open Issues](https://github.com/Bloxbind/Feedback/issues?q=is%3Aissue+is%3Aopen)
- [Closed Issues](https://github.com/Bloxbind/Feedback/issues?q=is%3Aissue+is%3Aclosed)

Searching helps us avoid duplicates and consolidate related discussions.

## Submit New Feedback

Choose the appropriate template for your submission:

- **[Feature Request](https://github.com/Bloxbind/Feedback/issues/new?template=feature_request.yml)** - Suggest a new feature or enhancement
- **[Bug Report](https://github.com/Bloxbind/Feedback/issues/new?template=bug_report.yml)** - Report a problem or issue
